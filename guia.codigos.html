Atributos de Identificación (Lo que conecta HTML con CSS) Estos no son etiquetas, sino "apellidos" que les pones a las etiquetas para que el CSS sepa a quién darle estilo: id="nombre": Se usa para identificar un elemento único en la página (como #hoja_cv o #foto_perfil). Solo puede haber uno con ese nombre. class="nombre": Se usa para identificar un grupo de elementos que quieres que se vean igual (como .titulo_lateral o .lista_puntos). Puedes usar la misma clase muchas veces. 8. Enlaces Externos (En el ) : Esta es fundamental. Sirve para "llamar" a tu archivo de CSS o a librerías externas (como los iconos de Bootstrap). Atributos de Imagen (Detalles importantes) En tu etiqueta , hay otros atributos que podrías estar usando: width y height: Para definir el tamaño directamente en el HTML (aunque es mejor hacerlo en CSS). title: Es el texto que sale en un globito cuando dejas el mouse encima de la foto. 1. Medidas de Tamaño width (Ancho): Define qué tan ancha es una caja. En tu #hoja_cv lo pusiste en 850px para que parezca una hoja real. height (Alto): Define la altura fija. min-height (Altura mínima): Es muy útil. Le dice a la hoja: "Mide al menos 1200px, pero si tengo mucho texto, estírate más". Así el contenido nunca se sale de la hoja. 2. Ubicación y Espacios (Cajas) padding (Espacio interno): Es el aire que hay dentro de la caja. Sirve para que el texto no toque los bordes. margin (Espacio externo): Es el espacio fuera de la caja. Sirve para separar una sección de otra. box-sizing: border-box;: Esta es técnica pero importante. Hace que el padding no "infle" el tamaño de la caja. Si dices que mide 100px, medirá 100px exactos aunque le pongas mucho padding. 3. Posicionamiento (Dónde va cada cosa) position: relative;: Se le pone al contenedor (la hoja). Es como decir: "Esta es el área de juego". position: absolute;: Se le pone a la foto. Permite moverla con coordenadas exactas (top y left) ignorando a los demás elementos. top / bottom / left / right: Son las coordenadas para mover elementos que tienen position: absolute. 4. Detalles de la Foto border-radius: 50%;: Convierte cualquier cuadrado en un círculo perfecto. object-fit: cover;: Es fundamental para fotos de perfil. Si la foto es rectangular, la recorta para que llene el círculo sin verse "aplastada" o estirada.
